## About The Project
An educational project dedicated to a testing

## Built With
Java 11
Maven 3.8.1
Junit 4.13.2
Mockito 5.3.1
Jacoco 0.8.10

## HowTo
Run project:
```shell
mvn clean test
```
Prepare jacoco test coverage report:
```shell
mvn verify
```
